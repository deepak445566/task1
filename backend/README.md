# 📝 Task Management App

This is a full-stack MERN project for managing tasks.

---

## 🚀 Features
- Add task
- Edit task
- Delete task
- Mark as completed
- Filter tasks (all/pending/completed)

---

## 🛠️ Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

---

## ⚙️ Setup Instructions

### Backend
```bash
cd backend
npm install
npm start
